# Software-Company-Network-Simulation
Cisco Packet Tracer implementation of a **secure multi-floor enterprise network** for a Software Service company with VLANs, DHCP, DNS, ACLs, RIP/OSPF routing, and DMZ staging environment.

## Project Objectives
- **Floor 1**: Separate DHCP for Developers/Business Analysts (RIP v2)
- **Floor 2**: VLAN isolation for Test Engineers/UI Designers + shared resources (OSPF)
- **Floor 3**: Secure staging web server with NAT/ACLs for external demos
- **Central**: DNS (192.168.0.13), Production server, Syslog
